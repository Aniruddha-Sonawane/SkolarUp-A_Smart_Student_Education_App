# SkolarUp â€“ Student Learning App

SkolarUp is a simple and student-focused mobile application built using React Native and Expo.
The aim of the project is to bring books, notes, posts, reports, and study-related tools together in one place, with a clean and minimal interface.

This project helped me understand real-world app structure, routing, component reuse, and managing multiple screens in a React Native environment.

---

## Features

- Book and notes viewer  
- Section for posts and updates  
- Reports screen  
- In-app WebView for opening study links  
- Basic chat prototype  
- Simple and easy-to-navigate UI

## Purpose of the Project

SkolarUp was created as a learning project to improve my skills in React Native development.
The goal was to build a functional mobile application with clear structure, reusable components, and smooth navigation.
This project reflects my interest in building practical tools for students and improving my mobile development skills.

---

## CI Pipeline

The project includes a basic GitHub Actions workflow to automatically run TypeScript checks, linting, and a build check on every push to the main branch.
This keeps the codebase clean and avoids broken builds.
